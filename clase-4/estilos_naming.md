## Estilos de nombramiento

Existen varios estilos para nombrar, variables, constantes, funciones, programas, clases...

1. Snake case:
```c
    int desc_principal;
    int desc_intermedio; 
    // Variables
```
2. Pascal case: (Upper camell case)
```c
    int MasaSuperior; // Clases
```

3. Lower camell case:
```c
    int masaSuperior; // Funciones
```
